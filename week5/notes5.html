strict mode

ECMAScript 5 introduced a strict mode that produces more exceptions and warnings and prohibits the use of some deprecated features. Strict mode encourages a better quality of JavaScript to be written.

Added to the first line of a JavaScript file: 'use strict';

Linting tools

Linting tools such asJS Lint,JS Hint,andES Lintcan be used to test the quality of JavaScript code, beyond simply using strict mode. Linting tools are also useful for enforcing a programming style guide.
