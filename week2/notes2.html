Programming basics

Variables

Variables are used in programming languages to refer to a value stored in memory.

Primitive Data Types

  1. String
  2. Number
  3. Symbol
  4. Boolean
  5. Undefined
  6. Null

*Any value that isn’t one of the primitive data types listed above is an object. (example - arrays, functions and object literals)

Scope

  1. Global scope

    2. It is accessible everywhere in the program

  2. Local scope

    3. This means that any variables defined inside a block using the let or const will only be available inside that block and not be accessible outside of that block.

Template Literals

A special types of string that were introduced in ES6. Template literals use the backtick character, ` , to deliminate the string. This means that a JavaScript expression can be inserted inside a string and the result will be displayed. Template literals can also contain line breaks, which are all converted into a Line Feed (\n)

Hard equality

NaN is the only value in JavaScript that is not equal to itself
