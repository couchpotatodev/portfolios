Transforms

The CSS3 transform property lets you lets you translate, rotate, scale, and/or skew any element on the page.

Translation

Translation functions allow you to move elements left, right, up, or down.

Scaling

The scale(x,y) function scales an element by the defined factors horizontally then vertically. If only one value is provided, it will be used for both the x and y values, growing or shrinking your element or pseudo-element while maintaining the original aspect ratio.

Rotation

The rotate() function rotates an element around the point of origin by a specified angle value.

Skew

The skew(x,y) function specifies a skew along the x and y axes.

Steps to create a simple transition using only CSS

  1. Declare the original state of the element in the default style declaration.
  2. Declare the final state of your transitioned element; for example, a :hover state.
  3. Include the transition functions in your default style declaration using the transition properties, including: transition-property , transition-duration , transition-timing-function , and transition-delay

The transition-duration Property

The transition-duration property sets how long the transition will take: the duration of time it takes to go from the default state to the transitioned state. You can specify this either in seconds ( s ) or milliseconds ( ms ).

The transition-timing-function Property

The transition-timing-function lets you control the pace of the transition in even more granular detail. The most common timing functions include the key terms ease , linear , ease-in , ease-out , or ease-in-out

Canvas

With HTML5’s Canvas API, we can draw anything we can imagine, all through JavaScript. This can improve the performance of our websites by avoiding the need to download images off the network. With canvas, we can draw shapes and lines, arcs and text, gradients and patterns.

The Canvas API is supported in:

  • Chrome 4+
  • Firefox 2+
  • Opera 9.6+
  • Safari 3.1+
  • iOS 3.2+
  • Internet Explorer 9.0+
  • Android 3.0+

XML

XML stands for eXtensible Markup Language. It’s a system meant to annotate text. Just as we can use HTML tags to wrap our content and give it meaning, so can XML tags be used to describe data, such as the content of a file.

The viewBox attribute

It defines the starting location, width, and height of the SVG image.

Drag and Drop

This API allows us to specify that certain elements are draggable, and then specify what should happen when these draggable elements are dragged over or dropped onto other elements on the page.

Drag and Drop is supported in:

  • All recent versions of Chrome, Firefox, and Safari
  • Partial support in Internet Explorer 7.0+ (IE10 doesn’t allow setting the drag image, but supports all other features)
  • Opera 12+
