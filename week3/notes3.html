Programming basics

Variables

Variables are used in programming languages to refer to a value stored in memory.

Primitive Data Types

  1. String
  2. Number
  3. Symbol
  4. Boolean
  5. Undefined
  6. Null

*Any value that isn’t one of the primitive data types listed above is an object. (example - arrays, functions and object literals)

Scope

  1. Global scope

    2. It is accessible everywhere in the program

  2. Local scope

    3. This means that any variables defined inside a block using the let or const will only be available inside that block and not be accessible outside of that block.

Template Literals

A special types of string that were introduced in ES6. Template literals use the backtick character, ` , to deliminate the string. This means that a JavaScript expression can be inserted inside a string and the result will be displayed. Template literals can also contain line breaks, which are all converted into a Line Feed (\n)

Hard equality

NaN is the only value in JavaScript that is not equal to itself

Arrays

An ordered list of values

Sets

An empty set is created using the new operator and Set() constructor

Maps

A convenient way of keeping a list of key and value pairs, and are similar to “hashes”, or“hash tables” or “dictionaries” in other programming languages.

Loops

  1. while loop
  2. do while loop
  3. for loop

Function

A chunk of code that can be referenced by a name, and is almost like a small, self-contained mini program. Functions can help reduce repetition and make code easier to follow.

Declaration

To define a function literal we can use a function declaration. This starts with the function keyword and is followed by the name of the function. ES6 introduced a new way to define functions, using what is known as “arrow” notation.

Invoking a function

To invoke a function, simply enter its name, followed by parentheses.

Parameters and Arguments

Parameters and arguments are terms that are often used interchangeably to represent values provided for the function as an input.

There is a subtle difference though: any parameters a function needs are set when the function is defined. When a function is invoked, it is provided with arguments.

Arrow Function

Arrow functions can be identified by the arrow symbol, => that gives them their name. The parameters come before the arrow and the main body of the function comes after. Arrow functions are always anonymous, so if you want to refer to them, you must assign them to a variable.

Callbacks

A function that is passed as an argument to another is known as a callback.
